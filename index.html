核心理念：三要素驾驭 Agent

Agent 不是来替代你的判断力的，而是来放大你的执行力的。驾驭 Agent 的关键在于三要素——设计好任务、管理好信息、设定好规则。

📝

Prompt

结构化任务指令书
告诉 Agent 做什么、给谁看、输出什么格式

→
📂

Context

Agent 的视野与记忆
管理它能看到什么、知道什么、记得什么

→
🔒

Harness

约束护栏与校验机制
防止幻觉、跑偏，确保输出可控可信

对比维度 传统 ChatGPT AI Agent（Claude Code）
交互方式 一问一答，人驱动 多步骤自主执行，人设定目标
能力边界 只动嘴（生成文字） 能动手（读写文件、运行代码、调用工具）
使用门槛 需要人持续输入 一次设定，自主完成复杂任务链
错误模式 说错话（幻觉） 做错事（文件覆盖、数据误读），需人工校验

你将收获什么？

工作坊结束后，你将带走三个核心认知

1

Agent 是杠杆，不是替代

它放大你的执行力，不替代你的判断力。Agent 可以帮你写初稿，但签字的是你。

2

质量 80% 取决于 Prompt

15% 取决于 Context，5% 取决于模型。不要迷信更贵的模型，先优化你的指令。

3

永远假设 Agent 会犯错

边界约束、校验机制、人工复核——三道防线缺一不可。最终守门人：人（你）。

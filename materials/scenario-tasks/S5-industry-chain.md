# S5: 产业链图谱绘制

> **类型**：产业
> **难度**：★★★
> **建议时长**：90分钟
> **适合人群**：有产业分析、供应链、战略背景

---

## 背景设定

你是某咨询公司产业研究部的顾问。你的客户（一家汽车零部件企业）计划进入新能源汽车电池领域，需要快速了解**动力电池产业链**的格局。

你的任务是：基于提供的材料，绘制一份**产业链图谱**，帮助客户理解"谁在哪个环节、谁和谁有关系"。

---

## 任务目标

基于提供的模拟材料，完成以下工作：

1. **产业链分层梳理**（上游→中游→下游）
2. **关键玩家识别**（每个环节的头部企业）
3. **关系图谱绘制**（用文本版 Mermaid 图或表格呈现）
4. **进入壁垒分析**（客户如果进入，卡在哪个环节？）

---

```

- 电解液：天赐材料、新宙邦

中游（电池制造）：
- 电芯制造：宁德时代、比亚迪、LG新能源、松下、三星SDI
- 模组/PACK：宁德时代、比亚迪、普莱德（北汽系）

下游（应用）：
```

### 材料B：某客户的背景

```
【客户背景：某汽车零部件企业】

现有业务：
- 汽车座椅骨架、内饰件制造
- 年营收约15亿元
- 主要客户：某传统车企集团（占比60%）、某日系车企（25%）、其他（15%）

进入动机：
- 传统汽车零部件增长放缓，需寻找第二增长曲线
- 现有客户（传统车企）正在转型新能源，有电池PACK外包需求

资源与短板：
- 优势：制造能力、客户关系、质量管理体系（IATF 16949认证）

- 预算：首期投资约2亿元
```

## 输出要求

### 格式要求

1. **产业链分层图**（文本版，可用缩进或表格呈现）
   ```
   上游：原材料
   ├── 锂矿/锂盐：天齐锂业、赣锋锂业...
   ├── 正极材料：...
   └── ...
   ```

2. **关键玩家表格**（每个环节的Top 3-5家，附市场份额或地位说明）

3. **关系图谱**（Mermaid flowchart 或 文字描述，展示"谁给谁供货"）

4. **进入壁垒分析**（针对客户的资源与短板，分析进入每个环节的难度）

5. **建议进入路径**（2-3条可选路径，附优劣势）

### 约束要求

- 只基于提供的材料，不补充外部企业信息
- 对无法确认的市场份额，标注[估算]
- 进入壁垒分析需基于客户的具体资源与短板
- 完成后列出2个产业链中信息最不充分的环节

## 建议的 Prompt 框架

```
【Context】
我们是某咨询公司产业研究部，客户是一家汽车零部件企业
（年营收15亿，主营座椅骨架/内饰件，有精密制造能力）。
客户计划进入动力电池领域，首期预算2亿元。
已上传产业链概述和客户背景。

【Objective】
基于提供的材料，绘制动力电池产业链图谱，
并为客户分析进入路径。

【Style】
管理咨询顾问风格，结构化、框架化、洞察驱动。

【Tone】
客观、务实，承认信息局限。

【Audience】
客户企业的战略委员会（CEO、CFO、战略VP），
他们熟悉汽车行业但对电池产业链了解有限。

【Response】
1. 产业链分层图（缩进文本或表格）
2. 关键玩家表格（每环节Top 3-5家）
3. 关系图谱（Mermaid或文字描述）
4. 进入壁垒分析（基于客户资源与短板）
5. 建议进入路径（2-3条，附优劣势）

【Constraint】
- 只基于提供的材料
- 市场份额标注[估算]
- 进入壁垒需结合客户具体情况
- 完成后列出2个信息最不充分的环节
```

---

## 评价要点

| 维度 | 重点 |
|------|------|
| **Prompt** | 是否要求了Mermaid图？是否定义了客户背景？ |
| **Context** | 是否综合使用了产业链概述和客户背景？ |
| **Harness** | 是否有"不补充外部信息"的约束？进入壁垒是否基于客户资源？ |
| **输出质量** | 图谱是否清晰？进入路径建议是否可行？ |
| **迭代过程** | 是否发现了产业链中的复杂关系（如交叉持股）？如何处理？ |

---

## 提示与技巧

### 可能的"翻车"点

1. **Agent 可能混淆层级**：把"正极材料"和"电芯制造"混为一层
2. **Agent 可能遗漏关系**：只列玩家，不展示"谁给谁供货"
3. **Agent 可能脱离客户背景**：给出通用的进入建议，不考虑客户的2亿预算和无电芯技术

### 如何提升质量

- 明确要求"用缩进格式展示层级关系"
- 要求"每个环节列出Top 3玩家，附一句话地位说明"
- 在 Prompt 中强调"所有建议必须考虑客户的资源与短板"

### 可扩展的方向

- 增加"竞争强度评估"（每个环节的竞争激烈程度1-5星）
- 增加"利润池分析"（哪个环节最赚钱？）
- 增加"时间轴"（从进入到盈利预计需要多久？）
